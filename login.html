Desenvolvimento da Página de Login

A página de login foi desenvolvida utilizando as seguintes tecnologias:

  • HTML: Estrutura básica da página, com a criação dos formulários de login e cadastro. Utilizando tags como `form`, `label`, `input`, etc.
  • CSS: Estilização da página, utilizando Bootstrap para a responsividade e um design moderno. Classes como `d-flex`, `justify-content-center`, `form-control` e `btn` foram utilizadas para formatar os elementos.
  • JavaScript: Interação com o usuário, incluindo a função `login()` para validar as credenciais de acesso (implementada de forma simplificada neste projeto). Também inclui a função `cadastro()`, que simula a criação de um novo usuário (sem conexão com um banco de dados).
  • JQuery: Biblioteca JavaScript utilizada para manipular o DOM (Document Object Model) de forma mais eficiente. É utilizada no esconder/mostrar os formulários de login e cadastro, e também para facilitar a manipulação do botão "Voltar ao Topo".
